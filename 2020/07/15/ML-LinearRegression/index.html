1.Theory

Step 1: Define Function

For given data set, we need to define a function to describe the relationship between input and output.

fw,b(x)=∑inWiXi+bf_{w,b}(x)=\sum_{i}^nW_iX_i+b fw,b​(x)=i∑n​Wi​Xi​+b

W is weight, b is intercept. For a simple example, y=wx+by=wx+by=wx+b

Step 2: Cost Function

Cost function is to measure the quality of our defined function. Here we use MSE(Mean Square Error)

L(f(x))=1N∑i=1n(f(xi)^−yi)2L(f(x))=\frac{1}{N}\sum_{i=1}^n(\hat{f(x_i)} -y_i)^2 L(f(x))=N1​i=1∑n​(f(xi​)^​−yi​)2

Notes

  • N is total number of data
  • f(x)^\hat{f(x)}f(x)^​ is estimated output by our defined function, yiy_iyi​ is the actual output
  • L(f)L(f)L(f) is the loss function

Step3: Optimal Function-Gradient Descent

How to choose the best function? In other words, how could we choose the best parameter w,b ?

If we can get minimum cost, that means we find the best function. To achieve it, we need to calculate partial derivatives of parameters to decide the direction of update and learning rate to determine how much should we update.

w←w−η∗dL(f(x))dww\gets w - \eta *\frac{dL(f(x))}{dw} w←w−η∗dwdL(f(x))​

b←b−η∗dL(f(x))dbb \gets b - \eta *\frac{dL(f(x))}{db} b←b−η∗dbdL(f(x))​

Notes:

  • η\etaη is learning rate
  • normally, w and b could be put in one vector W=[wb]W={w \brack b}W=[bw​]
  • derivatives could be ∇L=[dwdb]\nabla L={dw \brack db}∇L=[dbdw​]
  • thus, simplify it as W=W−η∗∇LW=W-\eta * \nabla LW=W−η∗∇L
  • when derivates are equal to 0, stop update.
